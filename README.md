# CHIP-8 Emulator in C# (Raylib)

A fully working CHIP-8 emulator built from scratch using C# and Raylib.

## Features
- 4KB memory system
- 16 registers + I register
- Stack-based subroutine handling
- Full CHIP-8 instruction set support
- 64x32 monochrome display
- Keyboard input mapping
- Delay & sound timers
- Sprite rendering with collision detection
- Built-in font set
- ROM loader support
- Audio beep when sound timer is active

## Controls
Mapped to CHIP-8 keypad:
1 2 3 4  
Q W E R  
A S D F  
Z X C V  

## How to Run
1. Install .NET
2. Install Raylib-cs
3. Place ROM file in project folder
4. Run Program.cs
